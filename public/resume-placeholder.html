Resume Placeholder

This is a placeholder for your actual resume PDF file.

How to add your actual resume:

  1. Create your resume as a PDF file
  2. Name it resume.pdf
  3. Place it in the public folder of your project
  4. The download link in your portfolio will automatically use this file

Once you've added your resume.pdf file, this placeholder will be replaced with your actual resume when users click the download button.

Note: In a production environment, you should always use your actual resume PDF file instead of this placeholder.
